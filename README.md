# Library Management System

A Java-based Library Management System developed using **Eclipse IDE**. This project demonstrates Object-Oriented Programming (OOP) concepts and provides basic functionality for managing books and library operations.

## Features

- Add new books
- View available books
- Search books by title or ID
- Issue books
- Return books
- Manage library records
- Input validation and exception handling
- Console-based application

## Tech Stack

- Java
- Eclipse IDE
- Git & GitHub
